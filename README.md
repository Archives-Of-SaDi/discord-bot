# Discord bot Example

```bash
$ git clone --depth 1 https://github.com/SaDi-BRo/discord-bot.git

$ cd ./discord-bot/

$ npm install

$ cp ./.env.example .env
```

<i>after setting your env vars</i>

```bash
$ npm run start
```

Enjoy!
